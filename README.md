  <h1 align="center">WalletWise - Modern Finance Manager</h1>
</p>

### Modern Finance Manager web application based on Django

### Features:
- Modern and sleek Home dashboard with quick summaries for incoming, outgoing transactions, balance and recent transactions
- A calendar used for easier date management
- A pre defined set of categories where every category shows the number of items it has and the value of those items
- Custom tags created by the user with similar functionalities like categories
- Separate screens for incomes and expenses which show the date, the amount and the tags associated with the items
- Adding, Editing and Deleting of tags, expenses and incomes
- Modern Log In and Register pages
- Beautiful animations that complete the project
